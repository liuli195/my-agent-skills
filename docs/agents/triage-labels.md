# Triage Labels

这些 engineering skills 使用五类固定的 triage 角色。本文件记录这些角色在当前仓库 issue tracker 中对应的实际标签名。

| Canonical role | 本仓库标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者判断 |
| `needs-info` | `needs-info` | 等待提交者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，agent 可以接手 |
| `ready-for-human` | `ready-for-human` | 需要人来实现 |
| `wontfix` | `wontfix` | 不处理 |

当 skill 提到某个 triage 角色时，使用上表中的对应标签。
